Now what exactly is periodicity, it is the tendency of a particular phenomena to repeat at regular intervals. Periodicity is associated with symmetry.

In the case of The Fourier Transform we generally talk about periodicity in space and time. We can write any signal in terms of sinusoids because they are simple and periodic in nature. Let‚Äôs take a variable ‚Äút‚Äù which will represent the time domain, then sin(2œÄt) is also periodic. If we fix the period of a signal to be 1 then the sum of the signals must also have the same period because the whole pattern cannot repeat unless the longest pattern repeats. So let‚Äôs take the sum of signal.

\[ \sum_{k=1}^{n} A_k sin (2\pi kt) cos (\varphi_k) \]

We can rewrite this as :

\[ \sum_{k=1}^{n} a_k cos {2\pi kt} + b_k sin {2\pi kt} \]

This is the sum of the signal in sines and cosines; however, we usually write the expression of the sum in complex coefficients. This is because it is better, algebraically and conceptually, to write it in this form.

\[ e^{2\pi ikt} = cos{2\pi kt} + isin{2\pi kt}\]

This is Euler‚Äôs formula. Now you may realize that our expression looks really different from this formula; however, through simple algebra we can get the following values for cosine and sine.

i = ‚àö-1

\[ cos {2\pi kt} = \frac{e^{2\pi ikt} + e^{-2\pi ikt}}{2}\]

\[ sin {2\pi kt} = \frac{e^{2\pi ikt} - e^{-2\pi ikt}}{2i}\]

(I am not showing you the derivation because it‚Äôs extremely simple and it would be better if you do it yourself; however, if you still have any doubts feel free to drop me an email and I will answer your queries.)

So in this way we can convert a trigonometric sum to the form :

\( f(t) = \sum_{k = -n}^{n} D_k e^{2\pi ikt} \) where f(t) represents a real signal

Here, we take the summation from ‚Äú-n‚Äù to ‚Äún‚Äù because we are only considering real signals, and the \(D_{k}\) s are the complex coefficients. If you will write the expression and make the substitution in terms of the complex exponentials for the coefficients you will find symmetry among them. You should try this yourself it‚Äôs really interesting and fun to do! When you do this you will find that all the complex terms essentially get cancelled and we get a real function. This is because we took the summation from ‚Äú-n‚Äù to ‚Äún‚Äù.

Here, the complex numbers will satisfy the symmetry property :-

\[D-n = D_{n}^{*} \]

Let‚Äôs say we have a periodic function, can we write it as a trigonometric sum / can we express it in terms of sinusoids?

In order to answer this question let‚Äôs suppose we can write the equation and then find the unknown variable. Here the unknown variable is Dk so let‚Äôs take any mth term and write it in a way by which we can take the unknown term out and put it on the other side, as follows:-

\[ f(t) = D_{-n}e^{-2\pi int} ‚Ä¶‚Ä¶‚Ä¶ D_me^{2\pi imt} ‚Ä¶‚Ä¶‚Ä¶. D_{n} e^{2\pi int}\]

\[ D_{m}e^{2 \pi imt} = f(t) - \sum_{k \not = m}D_{k}e^{2 \pi ikt} \]

I separated the mth term and then got this equation.

Let‚Äôs take the exponential term to the other side of the equation.

\[ D_{m} = e^{-2\pi imt} f(t) - D_{k} e^{2\pi i(k-m)t} \]

Now we will integrate this equation from 0 to 1 on both sides.

\[ \int_{0}^{1} D_{m} \sdot dt = \int_{0}^{1} e^{-2\pi imt}f(t) \sdot dt - \sum_{k \not = m}D_{k} \int_{0}^{1} e^{2\pi i(k-m)t} \sdot dt \]

\[ D_{m} = \int_{0}^{1} e^{-2\pi imt}f(t) \sdot dt - \sum_{k \not = m} D_{k} \frac{1}{e^{2\pi i(k-m)}} e^{2\pi i(k-m)t} ]_{0}^{1} \]

Now let‚Äôs evaluate the last term from 0 to 1. When t = 0, only the fraction part is left because \( e^{0} = 1\). When t = 1, we get \(e^{2 \pi i(k-m)}\). ‚Äú(k-m)‚Äù is an integer so we essentially have \(e^{2 \pi i}=cos(2\pi)+isin(2\pi).\)

\(sin(2 \pi) = 0\) and \(cos(2 \pi) = 1\), so we again have the fraction part left as follows :

\[ D_{m} = \int_{0}^{1} e^{-2\pi imt}f(t) \sdot dt - \sum_{k \not = m} D_{k} (\frac{1}{e^{2\pi i(k-m)}} - \frac{1}{e^{2\pi i(k-m)}}) \]

Therefore, the last term results to 0

Congratulations !! ü•≥ You have got an equation to calculate the complex coefficient in the equation of the periodic function that we considered.

\[ D_k = \int_{0}^{1} e^{-2\pi ikt} f(t) \cdot dt\]

(I have made a tiny change in this equation : replacing ‚Äúm‚Äù with ‚Äúk‚Äù.)

Thus, in order to write a general periodic function of the form \( f(t) = \sum_{k = -n}^{n} D_k e^{2\pi ikt}\) , the coefficient ‚Äì \( D_k\) ‚Äì has to be given by the formula :

\[ D_k = \int_{0}^{1} e^{-2\pi ikt} f(t) \cdot dt \]
